# codex-runner

Вспомогательная CLI-утилита для Windows, которая запускает codex с помощью runas.

## Установка

1) Обновите pip
```bash
python -m pip install --upgrade pip
```

2) Установите проект ГЛОБАЛЬНО из корня репозитория:

```bash
pip install -e .
```


## Запуск

После установки единожды потребуется указать имя пользователя, под которым запускается codex

```bash
codex-run init
```

После указания пользователя команда будет доступна как:

```bash
codex-run
```

Команда запускает codex --cd "<текущая директория>" в сессии cmd.exe, созданной через runas.
